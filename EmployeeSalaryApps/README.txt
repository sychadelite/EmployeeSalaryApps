------------------------ Cara  Mengakses Employee Salary Apps ----------------------------------------------

1. Signup terlebih dahulu apabila belum memiliki akun, karena sistem login sudah menggunakan firebase.

2. Login dengan email dan password yang telah terdaftar, pastikan benar dalam <5 percobaan. apabila lebih dari itu maka tombol login tidak bisa ditekan dan user harus membuka ulang aplikasi Employee Salary Apps.

3. Forgot Password dapat digunakan ketika user lupa passwordnya, email reset password akan dikirim dan user dapat menggantinya setelah mengakses link reset password pada email.
